# Fashion Recommendation System

A simple AI-based Fashion Recommendation System that suggests visually similar clothing items using image similarity.

## Features
- Image-based fashion recommendations
- MobileNetV2 feature extraction
- Cosine Similarity matching
- Fast and lightweight
- Easy to use

## Technologies Used
- Python
- TensorFlow / Keras
- MobileNetV2
- NumPy
- OpenCV
- Matplotlib
- Scikit-learn

## How to Run

1. Install required libraries:
pip install tensorflow numpy opencv-python matplotlib scikit-learn tqdm

2. Run the project:
python fashion_recommendation.py

## Output
The system takes a fashion image as input and displays the top 5 visually similar fashion items from the dataset.
